我的視線落在櫃台旁的傘架上。那裡插著一把看起來已經放很久的透明長柄傘，柄上甚至有點灰塵。

主角(心聲): 這種傘，大概是很久以前某個客人遺忘的吧。

@show 店長 left normal
主角: 「店長，這把傘……」

店長從報紙後抬起頭，看了一眼。

店長: 「喔，那個啊，放半年了。應該是沒人會來認領了。」

主角: 「我可以借嗎？下次還。」

店長聳聳肩，笑了一下。

@show 店長 left smile
店長: 「反正放著也是放著。」

@hide 店長 fade=400

我拿起那把傘，感覺比預期的沉一點。傘布有點舊，但撐開應該還能用。

我走向門口，但在最後一刻停下。

@show 小鈴 right normal
回頭，她還坐在原位，膝上攤著那本書。

主角: 「這個……」

我把傘放到她旁邊的椅子上。

主角: 「給妳。」

小鈴: 「欸？」

她愣住了。

:: if chose_bookstore
主角: 「我剛剛其實也是跑進來躲雨的。所以知道淋濕的感覺不好受。」
:: else
主角: 「我剛剛在對街躲了很久才跑過來。所以……」

我不知道該怎麼接下去。

主角: 「……所以，妳不要像我那樣。」
:: endif

主角: 「這把傘不是我的，是店長說沒人要的。但妳拿去用吧，不然我走了之後，妳要怎麼辦？」

@show 小鈴 right shy
小鈴: 「可是你也沒有傘啊？」

主角: 「我約會的地方有騎樓，跑一下就到。」

這當然是謊話。但我說完之後，也就只是笑笑。

她低頭看著那把傘，手指輕輕碰了碰傘柄。

:: if talked_about_ritual
小鈴: 「……這樣子，下次雨天來，我要怎麼跟店長解釋這把傘的下落？」

主角: 「就說……有個路過的人把它借走了。」

小鈴: 「借走？」

主角: 「嗯。所以我還要再來一次。」
:: else
小鈴: 「那你下次來拿回去？」

主角: 「嗯，再來一次。」
:: endif

她笑了。很輕，但我看到了。

@show 小鈴 right smile
小鈴: 「那……我會好好看著它的。」

主角: 「謝謝。」

我推開門。銅鈴響了。

@bg street_rain_evening.svg fade=1200

雨依然很大。我把外套的領子拉起來，跑進雨裡。

:: if chose_bookstore
主角(心聲): 借了一把不屬於自己的傘，然後又把它留給另一個人。這大概是今天最浪漫的決定。
:: else
主角(心聲): 我在對街猶豫了那麼久才走進那家店。結果走出來的時候，連傘都沒帶走。
:: endif

@wait 800

跑了半條街，我回頭看。

書店的燈光還在。

@wait 1500

主角(心聲): 那把傘留在那裡，就是一個未完成的故事。

主角(心聲): 而未完成的故事，總是要有人去完成的。

@wait 1500
