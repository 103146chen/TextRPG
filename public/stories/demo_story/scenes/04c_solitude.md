我對她揮了揮手。

@show 小鈴 right normal
小鈴: 「路上小心。」

主角: 「嗯。」

我推開門，銅鈴輕輕響了一下。冷風夾著雨水撲在臉上。

@bg street_rain_evening.svg fade=1500
@hide 小鈴 fade=400

我把外套的帽子拉起來，低頭走進雨裡。

:: if chose_bookstore
主角(心聲): 那家書店很溫暖。可惜，有些溫暖只是路過就好。
:: else
主角(心聲): 躲了那麼久的雨，現在又要走進去。有點好笑。
:: endif

走了幾步，我忍不住回頭看了一眼。書店的燈光透過雨霧，像一顆懸浮的暖黃色光點。

透過玻璃窗，我隱約看到她的身影，低著頭，好像在讀書——

也好像只是在等著什麼。

主角(心聲): 不是每一次相遇都要留下什麼。

:: if talked_about_ritual
主角(心聲): 她說那是她的儀式。那就讓我也成為她儀式裡的一個片段吧——一個淋濕的、經過的、沒有名字的人。
:: elif talked_about_books
主角(心聲): 她推薦的那本書，我大概永遠不會買了。有些東西一旦擁有，就不再是記憶裡那個樣子。
:: else
主角(心聲): 我甚至沒問她的名字。
:: endif

我把傘——不對，我根本沒有傘。我苦笑一聲，加快腳步。

雨越下越大。

@wait 1000

路燈亮了。我繼續往前走，沒有再回頭。

主角(心聲): 這個城市有太多擦肩。她會繼續來那家書店，我會繼續過我自己的日子。

主角(心聲): 或許若干年後，當我想起這個下午，會記得：有過一個書店，有過一場雨，有過一個安靜讀書的女孩。

主角(心聲): 而這樣就已經夠了。

@wait 2000
