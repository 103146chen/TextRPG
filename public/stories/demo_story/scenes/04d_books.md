主角: 「啊，對了——」

我停下腳步，回頭看她。

@show 小鈴 right normal
小鈴: 「嗯？」

主角: 「妳剛剛推薦的那本書，放在哪一區？我想……先看看。」

:: if talked_about_books
她眨了眨眼，似乎沒預期我會折返。
:: elif talked_about_ritual
小鈴: 「你不是有約？」

主角: 「還有時間。」

這不是真話。但我說出口的時候，自己也嚇了一跳。
:: else
小鈴: 「你不怕遲到嗎？」

主角: 「遲到一點還好。」
:: endif

她猶豫了一下，然後站起來。

@show 小鈴 right smile
小鈴: 「跟我來吧。」

@bg bookstore_interior.svg fade=600

她領著我走到書店最深處，那裡有一整面牆的台灣文學。她伸手從中間的架子上抽出一本，輕輕放到我手裡。

書的封面是一幅雨中街景的水墨畫。指尖觸到封面的瞬間，我感覺到她指尖的溫度還殘留在紙上。

小鈴: 「就是這本。」

主角: 「嗯……」

我低頭翻開。扉頁上有一行小字，是作者親筆簽的名——這本竟然是簽名本。

主角: 「這……」

:: if chose_bookstore
小鈴: 「舊書店有時候會有這種驚喜。」

主角: 「妳怎麼不買下來？」

小鈴: 「因為我每次來都想看一下它還在不在。」她頓了頓，補充：「這樣比擁有還踏實一點。」
:: else
小鈴: 「我每次雨天來，都會來確認它還在。」

主角: 「為什麼不自己收藏？」

小鈴: 「因為……一旦變成自己的，就不會這樣想念它了。」
:: endif

我看著她，突然覺得我懂了她剛才說的「儀式」。有些東西留在原處，才會一直珍貴。

主角: 「那我也不買。」

@show 小鈴 right shy
小鈴: 「？」

主角: 「我想讓它繼續留在這裡。下次再來，想確認它還在。」

她愣愣地看著我，然後笑了出來。那個笑容比剛才任何時候都更亮。

小鈴: 「那就是……我們的秘密了。」

主角: 「秘密？」

小鈴: 「一本我們都不買走的書。」

@wait 800

窗外的雨還在下，但我已經不急著走了。

主角(心聲): 有些相遇，不需要留下彼此的名字或聯絡方式。只需要一個共通的座標——這本書、這家店、每一個雨天的午後。

我們沒有再多說話，就這樣並肩站在書架前，各自讀著屬於自己的頁面。

@wait 1500
