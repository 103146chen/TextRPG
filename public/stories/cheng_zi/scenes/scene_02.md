你十六歲那年，阿母叫你辭去了長工。
母親: 「大了，找個更好的路子。」
贌幾畝田來種？製糖會社把好田都霸佔去了。業主見會社給的租金多，把地全交給會社，農民連種田的機會都沒有。
你要怎麼賺錢？

:: label after_job

:: if 少年工作 == "糖廠"
你進了製糖會社做日工。
天還沒亮就得進廠，天黑了才能出來。日本監工每隔一段時間就來走動，說話快，聲音大，你聽不大懂，但你看得懂他的眼神——你不是人，你是工具。
得參(心聲): 「有錢，是有錢的。但每天回到家，整個人都是空的。」
做了半年，你辭了。阿母也說: 「莫去了，人不該那樣被用的。」
@set 反抗積累 += 0
:: elif 少年工作 == "散工"
你做起散工——今天搬磚，明天拔草，後天挑水，哪裡有人喊就去哪裡。
比長工輕省，工錢又比以前多。你氣力大，做事勤，雇你的人家不少。
得參(心聲): 「東做一塊、西賺一角，累積起來也不少。」
阿母替你一點一點存下來，臉上慢慢有了點笑容。
@set 反抗積累 += 1
:: elif 少年工作 == "長工"
你找了個大富人家做長工。包吃包住，每月給固定工錢。
聽起來穩，但大戶人家看傭人的眼神，和日本監工看工人的眼神，其實差不了多少。
有一天主人罰了你一個月工錢——原因是菜弄壞了，但那棵菜其實不是你弄壞的。你站在那裡，沒有說話。
得參(心聲): 「算了。說了也沒用。」
@set 反抗積累 -= 1
@set 埋伏筆_第一次吞忍 = true
:: else
你知道做私菸是違法的，但村裡有人做，賺的比下田還多。
你跟著學了幾個月。
某天，查緝的人來了，同伴跑散了，你來不及跑，立在原地，對方走過來問你名字。
你心跳快到幾乎停掉，但你說: 「我是路過的。」
沙沙的草聲，然後他走了。
那之後，你再也沒有碰私菸。但你記住了一件事: 遇到那種時刻，腳是不受大腦控制的——只能靠命。
@set 反抗積累 += 2
@set 埋伏筆_嚐過恐懼 = true
:: endif
光陰似矢，容易地過了幾年。

母親: 「參仔，該成家了。」
你看著阿母，點了頭。
