你是秦得參。這是日本人統治台灣的時代，你住在彰化附近的一個村子裡。
這塊土地上的人都在討生活，而你家特別苦。
得參(心聲): 「阿爸死得早，我甚至沒看過他的樣子。」
阿爸在你出生前就過世了，留下的只有一間歪斜的土角厝。生前租來耕種的田，早已被地主收回。
@show 母親 center worried
阿母把你拉拔大。她白天畜養牲畜，晚上幫人洗衣服、縫補衣服，手上的繭厚得像是皺掉的樹皮。
阿母從沒讓你進過公學校——束脩要錢、書本要錢，家裡連下頓飯都不一定有著落。
母親: 「參仔，好好做工。做人莫怨嘆，怨嘆無路用。」
這是阿母最常說的話。
@hide 母親 fade=600
你從有記憶以來就在幫忙——放牛、撿柴、拔草、到田裡幫阿母。你不是特別聰明的孩子，也沒有什麼了不起的志向。你只是想，像阿母一樣，活下去。
@set 家境 = "極貧"
@set 個性 = "忍耐"
村子裡比你家有錢的人家不少，但像你家這樣苦的也有。大家都在同一個泥淖裡，看不出誰高誰低。
只是——有一天，村頭的林阿伯被帶走了。
得參(心聲): 「聽說是因為田租繳不出來。」
巡查大人帶著日本刀來的。林阿伯是全村有名的老實人，從來不與人爭。但那天他被帶走之後，就再也沒有回來。
阿母從那天開始，再也不讓你隨便出門。

:: if 童年印象 == "疑惑"
@show 母親 center worried
母親: 「憨囝仔，莫問。問也無用。」
她的眼神飄向遠方，像是看見了什麼你看不見的東西。
母親: 「這世間，有些事情無所謂對錯，只有『做得到』佮『做袂到』。」
@hide 母親 fade=800
你沒聽懂，但你記住了這句話。
:: elif 童年印象 == "畏懼"
那天晚上，你做了一個夢。夢裡有穿著黑衣服的人抓走了阿母，而你什麼都不能做，只能躲在田裡顫抖。
你醒來的時候，枕頭濕了一整片。從那天起，你變得更安靜了。
:: else
你沒有對任何人說。但那股火在你心裡燒著。
只是隨著時間過去，那把火慢慢變小、變暗。你漸漸明白——一個人的火，是燒不穿這個世界的。
至少你當時是這麼想的。
:: endif
歲月就這樣一年一年過去。你從一個小孩變成少年，從少年變成青年。
阿母的背越來越彎，你的肩膀越來越寬。
