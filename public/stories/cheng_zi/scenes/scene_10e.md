你把包袱收拾好，動作很輕。
孩子睡著了。阿秀睡著了。
你站在門口，看了很久。
@show 妻子 center normal
那張臉你看了那麼多年，皺紋多了，也瘦了。她睡著的樣子，比白天還要老。

:: if 少年工作 == "糖廠"
得參(心聲): 「從在糖廠做工，到現在，我也沒有讓你們好過多少。」
:: elif 埋伏筆_喪母空虛
得參(心聲): 「阿母走的時候，心裡空了一塊。現在那個空，好像更大了。」
:: endif
@hide 妻子 fade=400
@wait 1500
你轉過身，走出去了。
你往南走，沒有特別的方向，就是走。越過幾個村子，改做苦力，改名字，改什麼都改了。
沒有人再聽過秦得參的名字。
在彰化的市場，有人偶爾問起那個賣生菜的得參，沒有人知道。
@wait 1000
得參(心聲): 「我拋下了阿秀、孩子——」
得參(心聲): 「沒有同歸於盡，沒有壯烈。只有一個消失在殖民地邊緣的影子。」
@wait 2500
有一年，你在某個港口做挑夫。一個從彰化來的商人說，你長得和那個什麼秦得參很像。
你說，你不認識這個人。
那個商人說，哦，是個苦命的人，聽說他的妻子後來帶著孩子去了台南，也不知道有沒有活下去。
你搬起貨，頭也不回地走了。
@wait 2000
