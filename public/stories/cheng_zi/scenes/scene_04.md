那年早季收穫時，你染了瘧疾。
@show 妻子 center worried
妻子: 「參仔你燒起來了，先別去工作——」
得參: 「再去看一天，今天不去就沒有工錢。」
你強撐著去。到晚上回家，身體就垮了。
@hide 妻子 fade=400
三天後，高燒才退下來，但腳手乏力，肚子開始脹。
你怎麼治這個病？

?? choice
- text: 請西醫診治——花兩塊多錢，但藥效真的比較快
  set: { 治病方式: "西醫" }
  goto: after_sick
- text: 煎青草藥喝——不花錢，但效果慢
  set: { 治病方式: "青草" }
  goto: after_sick
- text: 什麼都不治，硬撐著繼續出門做輕工
  set: { 治病方式: "硬撐" }
  goto: after_sick
??

:: label after_sick

:: if 治病方式 == "西醫"
你花了兩塊多錢看了一次西醫。
得參(心聲): 「三天的工，還不夠吃一服藥……」
藥是有效，輕快了些。但那兩塊多，是你和阿秀省了大半個月的。
好容易退燒了，腳手仍然使不上力，你卻不敢閒著，又耐苦到田裡去。沒幾天，又發起來了。
就這樣反覆，拖了幾個月。
@set 反抗積累 += 0
@set 治病已負債 = true
:: elif 治病方式 == "青草"
你捨不得花錢請西醫，自己煎些不用錢的青草，或便宜的漢藥服食。
有人說未必無效，也有人說這樣吃壞了胃。
總隔兩三天發一回寒熱，拖了好幾個月，才不再發作。但肚子愈來愈脹。
有人說是青草吃太多所致，有人說是脾腫。
得參(心聲): 「不管什麼原因，只要不礙著出去做工就好。」
@set 反抗積累 -= 1
:: else
你什麼藥都沒吃，硬撐著繼續出門。
做不了重活，就做輕的——整理竹籬、餵雞，只要能動就動。
有幾天連輕活都做不了，你就坐在門邊曬太陽，看著阿秀抱著孩子出門。
妻子背影在巷口消失，你沒有出聲。
那些日子，孩子在床邊啼哭，你在榻榻米上呻吟，聲音交織，像是這個家發出的聲音。
@set 反抗積累 += 1
@set 埋伏筆_硬撐不治 = true
:: endif
時間總算過去了。
年末的時候，身體才能做些輕的活。
得參(心聲): 「看看尾牙快到了，新春也快了……家裡的糧食，從哪裡來？」
有人說鎮上賣生菜的生意不壞。你坐在昏暗的裡房，把這件事想了又想。
@set 病已過 = true
