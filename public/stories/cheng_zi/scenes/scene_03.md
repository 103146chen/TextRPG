你娶了阿秀。
@show 妻子 center normal
她是種田人家的女兒，不嬌氣，嫁進來之後就跟著你一起下田，挑水、插秧，一個人頂三個人用。
妻子: 「參仔，你母親那麼辛苦把你養大，我們要好好孝順她。」
得參: 「嗯。」
收成好的那幾年，日子過得還算過得去。
二十一歲，阿母懷裡抱著第一個孫子，臉上有了笑容。
@hide 妻子 fade=600
@show 母親 center normal
母親: 「我的心事，放下了。」
阿母的身體撐不住了。
病臥幾天，她臉上現著十分滿足的樣子，就這樣去了。
@hide 母親 fade=800
@wait 1500
可憐的得參，他的幸福，已和他慈愛的母親，一併失去。
站在阿母的床前，你心裡是什麼感覺？

?? choice
- text: 我要替阿母好好活下去，不讓她擔心
  set: { 面對喪母: "承擔" }
  goto: after_mama
- text: 心裡空掉了，說不清楚是什麼感覺
  set: { 面對喪母: "空虛" }
  goto: after_mama
- text: 一股說不清楚的氣在胸口——為什麼人活著這麼累？
  set: { 面對喪母: "憤怒" }
  goto: after_mama
- text: 記著她說的那句話：「怨嘆無路用。」
  set: { 面對喪母: "遺言" }
  goto: after_mama
??

:: label after_mama

:: if 面對喪母 == "承擔"
你跪在那裡，心裡默默立了個誓。
得參(心聲): 「阿母，我知道了。我會好好的。」
眼眶發熱，但你告訴自己不是時候，沒有讓眼淚掉下來。
@set 反抗積累 += 0
@set 埋伏筆_誓言承擔 = true
:: elif 面對喪母 == "空虛"
你跪著，說不出話，也哭不出來。
那種空，不像悲，更像是——什麼東西被抽走了，剩下一個還在動的軀殼。
旁邊的人說話、哭聲、鄰居進進出出，你都聽見，但都很遠。
@set 反抗積累 -= 1
@set 埋伏筆_喪母空虛 = true
:: elif 面對喪母 == "憤怒"
你想起她一輩子的辛勞。
想起她手上的繭，想起她說「莫怨嘆」，但她自己是在怨嘆什麼嗎？她沒有機會怨嘆，因為她從來沒有停下來。
你的手握緊了。
得參(心聲): 「這個世界讓人活得太辛苦了。」
@set 反抗積累 += 2
@set 埋伏筆_喪母憤怒 = true
:: else
你想起她最常說的那句話。
得參(心聲): 「怨嘆無路用……」
你不知道那是勸告，還是她自己學來對付這個世界的方法。也許兩種都是。
你把這句話收進心裡，像是把一件舊衣疊好放進箱底。
@set 反抗積累 -= 1
@set 埋伏筆_記住遺言 = true
@set 母親遺言已記 = true
:: endif
日子繼續走。
阿秀又生了一個女孩。家裡因為失去了阿母，事情全落在她身上——照顧孩子、料理三餐，不能再和你一起出去做工，進款少了一半。
你加倍工作，但身體開始撐不住了。
@set 成家已過 = true
