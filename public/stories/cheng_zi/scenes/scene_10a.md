元旦，參的家裡，忽然譁然，發生一陣叫喊、哀鳴、啼哭。
隨後，又聽著說: 「什麼都沒有嗎？」「只銀紙備辦在，別的什麼都沒有。」
@wait 2000
同時，市上亦盛傳著，一個夜巡的警吏，被殺在道上。
@wait 2500

:: if 反抗積累 >= 6
得參(心聲): 「那股火，我一直知道它在那裡。那晚，我不想再壓它了。」
你清楚地知道自己在做什麼。不是衝動，不是糊塗——是你這一輩子積下來的所有東西，在那個夜晚找到了出口。

:: if 童年印象 == "憤怒"
從林阿伯被帶走那天，那個誓願就在那裡了。
:: elif 埋伏筆_曾想反抗
那天在市場，你沒有動手。那一次，你動了。
:: endif
:: elif 反抗積累 >= 1
得參(心聲): 「我不知道我在做什麼。我只知道，手動了，腳就走出去了。」
那股東西——是憤怒嗎？是什麼？你說不清楚。
你一半清醒，一半不清醒地走向了那個夜晚。
也許這才是多數人的樣子: 不是英雄，也不是懦夫，只是一個被逼到某個臨界點的普通人。
:: else
得參(心聲): 「我不知道自己是怎麼走出去的。」
一輩子低頭，到最後那一刻，連你自己都不知道——是哪裡來的力氣，或者，根本沒有力氣，只是一種放棄了所有之後的、什麼都不重要了的飄移。
你沒有計畫，沒有憤怒，沒有清醒的意志。
你只是，走出去了。
賴和寫這個故事的時候說: 「凡強權行使的地上，總會發生這樣的事。」不是因為受壓迫的人是英雄，是因為把人逼到無路可走的地方，任何人都可能走向那個出口——哪怕他這一輩子都是最老實的那一個。
:: endif
@wait 2000
你沒有留下什麼話。
你知道阿秀哭過了，孩子啼哭了。
但你也記得，你的母親死時，臉上是滿足的——那是完成了的表情。
這幕悲劇，作者在很多年後寫下來，說: 「凡強權行使的地上，總會發生這樣的事。」
