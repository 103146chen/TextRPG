天色還未大亮，市上人聲早就沸騰了。
除夕的市場裡，到處都是採買年貨的人，擁擠熱鬧。年華垂盡，人生頃刻，使人感到說不清的悵惘。
到天亮後，各色貨擔多要完了。當這時候，你又遇到那個巡查。
@show 巡查 right angry
巡查: 「畜生，昨天跑哪兒去？」
得參: 「什麼？怎得隨便罵人？」
巡查: 「畜生，到衙門去！」
得參: 「去就去呢，什麼畜生？」
@hide 巡查 fade=400
巡查瞪了你一眼，把你帶上衙門去。
@wait 800
法官坐在上面，你跪在地下。
法官: 「汝秦得參嗎？」
得參: 「是，小人，是。」
法官: 「以前不管他，這回違犯著度量衡規則。科罰汝三塊錢，就算是格外恩典。」
得參: 「可是，沒有錢。」
法官: 「沒有錢，就坐監三天，有沒有？」
你在心裡打算: 新春閒時節，監禁三天不關係，還是三塊錢用處大。
得參: 「沒有錢。」
你就甘心去受監禁。
@wait 1000
才監禁下，就有人說有人替你繳了罰款。
出來到衙前，看見阿秀站在那裡，臉色蒼白。
@show 妻子 center worried
得參: 「妳為什麼到這兒來？」
妻子: 「聽說你被拉進去！」
她聲音有點哽咽。
得參: 「不犯到什麼事，不至殺頭，怕什麼。」
你們走到街上，市已散了，市上處處聽到辭年的爆竹聲。

:: if 上當鋪
得參: 「金花取回了沒有？」
:: else
得參: 「生意本還了沒？」
:: endif
妻子: 「還未出門，就聽到你的消息。我趕緊趕來，在那兒繳去三塊。現在還不夠。」
你恍然地嘆了一聲，把今早賺到的三塊錢拿給她。

:: if 上當鋪
:: else
得參: 「你先回家吧，我去去就回。」
:: endif
@hide 妻子 fade=400
@wait 1500
三塊罰款，三塊贖金——今天賺的全沒了。沒了，餘下新春用度亦不夠了。
這段時間欠缺的，怎麼補？

?? choice
- text: 去找臨時的零工——除夕這天，還有沒有人需要人手？
  set: { 籌款方式: "零工" }
  goto: after_bail
- text: 央求廟口老頭或岳家，看能不能再借
  set: { 籌款方式: "再借" }
  goto: after_bail
- text: 找出家裡還剩什麼，能變賣的就變賣
  set: { 籌款方式: "變賣" }
  goto: after_bail
- text: 不想了，讓它去。回家先陪孩子過年
  set: { 籌款方式: "放棄" }
  goto: after_bail
??

:: label after_bail

:: if 籌款方式 == "零工"
除夕這天確實有人還需要人手——搬年貨、打掃院子。
你跑了幾戶，做了半天，總算多賺了一塊多錢。
她看著你一日憔悴的樣貌，沒有說話。
@set 反抗積累 += 1
:: elif 籌款方式 == "再借"
你去了，但對方看見你除夕這天趕來，神情都有些猶豫。
廟口老頭家搖了頭說: 「不是不幫你，只是年底我自己也緊……」
你謝了他，回去了。沒有借到。
回家的路上，你試著告訴自己: 人已經盡力了。
@set 反抗積累 += 0
:: elif 籌款方式 == "變賣"
家裡能賣的，早就剩得不多了。
你翻了翻，找到幾樣——一個舊的銅鑄香爐，阿母留下的；幾件舊衣，汙黃間隔著破洞。
你捧著那個香爐，站了很久。
最後還是放了回去。有些東西，就算窮成這樣，也有些東西不能賣。
得參(心聲): 「算了。」
@set 反抗積累 -= 1
:: else
你什麼都沒做，就回家了。
孩子們在院子裡玩，看見你回來，跑過來拉你衣角。
你蹲下來，摸了摸他們的頭，沒有說話。
今天的事，先不想了。
@set 反抗積累 += 0
:: endif
圍過爐，孩子們因明早要早起開正，各已睡下，在做他們幸福的夢。
@set 除夕已過 = true
