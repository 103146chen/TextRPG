你回到家裡，夜飯前吃不下，悶悶地一句話不說。
@show 妻子 center worried
阿秀看出你不對勁，殷勤地問。你才把白天的事情說給她聽。
說到那桿斷掉的秤仔，說到那個帝身的帳子，說到那個「畜生」字眼。
妻子: 「寬心罷。這幾天的所得，買一桿新的還給人家，剩下的猶足贖取那金花回來。」
妻子: 「……只是今年運氣太壞，怕運裡帶有官符。經過這一回事，明年或許快就出運了。」
@hide 妻子 fade=400
你坐在那裡，聽著妻子說話，心裡是什麼感覺，你說不清楚。
這一次的事情，你心裡怎麼擺放？

:: label after_home

:: if 面對罰金 == "低頭"
你深吸了一口氣。
得參(心聲): 「算了。人在屋簷下，不得不低頭。」
這道理你懂。從小就懂。
但懂，不代表不痛。
你去睡了，但半夜醒了兩次，不知道為什麼。
@set 反抗積累 -= 2
:: elif 面對罰金 == "硬吞"
你什麼都沒說。
心裡卻已經算好了: 這次損失多少，下次要多賣幾天，才能補回來。
得參(心聲): 「已經這樣了，再想也沒有用。想辦法。」
這個「想辦法」三個字，你跟自己說了很多次，才讓心裡那股火壓下去。
@set 反抗積累 += 1
@set 埋伏筆_每次硬吞 = true
:: elif 面對罰金 == "想搶"
你坐在那裡，腦子裡一直有個畫面揮不去——
如果你當時伸手，把那桿秤搶回來。如果你說，你憑什麼打斷它？
但你知道那之後會怎樣。你在村裡有老婆孩子，你不能那樣做。
得參(心聲): 「……你就是因為有這些，才動不了手。」
這個想法讓你比剛才更難受。
@set 反抗積累 += 3
@set 埋伏筆_曾想反抗 = true
:: else
你想起了那句話。
得參(心聲): 「怨嘆無路用……」
阿母說這句話的時候，是什麼表情？是安慰你，還是她用了一輩子的時間說服自己的？
你不知道。
你也不知道這句話，到底對不對。

:: if 母親遺言已記
得參(心聲): 「阿母，你怎麼看待這樣的事情？」
:: endif
@set 反抗積累 -= 1
:: endif
休息過一天，看看沒有什麼動靜。明天就是除夕了，只剩最後一天的生意。
你安坐不住。
得參(心聲): 「就一天。賺了這最後一天，就能把事情都收尾了。」
你絕早起來，挑上菜擔，往鎮上去。
@set 回家已過 = true
