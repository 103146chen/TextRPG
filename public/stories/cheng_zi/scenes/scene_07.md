那天近午，一個下級巡查走到你的擔前，目光落在你的生菜上。
@show 巡查 right cold
你立刻挺直身體，臉上堆出笑來。
得參: 「大人，要什麼不要？我的菜今天特別新鮮。」
巡查: 「汝的貨色比較新鮮。」
巡查拿起一朵花菜，看了看。
巡查: 「稱稱看。」
你老實地依照要求，將花菜掛上秤仔，仔細稱了稱——不多不少，正好兩斤。

?? choice
- text: 「大人，真客氣啦！才一斤十四兩！」
  set: { 回答: "打折" }
- text: 「大人，正好兩斤！」
  set: { 回答: "原價" }
  goto: after_answer
??

:: label after_answer

:: if 回答 == "打折"
巡查: 「一斤十四兩？不錯罷？」
說著這話的同時，巡查瞄了一眼秤上的刻度，臉色驟變。
得參: 「不錯，本有兩斤足，因為是大人要的......」
巡查: 「這桿秤仔，不對。」
:: else
巡查: 「兩斤？你這秤子看來不堪用了。」
巡查瞄了一眼秤上的刻度，板著臉說。
:: endif
你知道他說的不是事實。這桿秤仔是鄰居借的，新新的，哪裡有問題？
面對這個指控，你怎麼回應？

?? choice
- text: 恭敬地說「大人說得是，小人不懂，請大人指點」
  set: { 面對巡查: "老實" }
  goto: after_weigh
- text: 裝傻，說「啊，我也不太會用，可能是小人笨」
  set: { 面對巡查: "裝傻" }
  goto: after_weigh
- text: 泰然地說「這桿還新新呢，大人要不信可以再驗」
  set: { 面對巡查: "辯解" }
  goto: after_weigh
- text: 把花菜湊過去：「大人，這些就送您，請笑收」
  set: { 面對巡查: "送菜" }
  goto: after_weigh
??

:: label after_weigh
@show 巡查 right angry

:: if 面對巡查 == "老實"
你老老實實地低下頭。
巡查看著你服從的樣子，反而眼睛裡閃過一絲輕蔑。
巡查: 「不懂還拿出來用？沒見識的東西。拿過來！」
他把秤仔接過去，翻看了幾下，說: 「不堪用了，拿到警署去！」
得參: 「什麼緣故？」
巡查: 「哪有那麼多為什麼！畜生！」
啪的一聲——他把秤仔打斷，擲在地上，掏出小帳子，把你的名字、住處記下，走了。
@set 反抗積累 -= 2
:: elif 面對巡查 == "裝傻"
你陪笑，擺出一副自己也很疑惑的樣子。
得參: 「是啊，小人也不太懂，這是跟鄰居借的，不知道有沒有差。」
巡查盯了你一眼——他不相信你，但也抓不到把柄。
片刻沉默。你以為他要放過你了。
然後他說: 「正因為借的才更有問題，借的怎麼知道是不是準的？拿來！」
啪的一聲——他把秤仔打斷，擲在地上，掏出小帳子，把你的名字、住處記下，走了。
得參(心聲): 「裝傻，也沒有用。」
@set 反抗積累 -= 1
:: elif 面對巡查 == "辯解"
你泰然地說。
得參: 「不，還新新呢！大人若要驗，小人願意配合。」
巡查的臉色更難看了。
巡查: 「汝是要教我怎麼做事嗎？」
他的語氣升高了半個調。周圍的人都往旁邊挪了挪。
巡查: 「拿過來！」
啪的一聲——他把秤仔打斷，擲在地上。這次力道更大，好像在打的不只是秤，還有你說的那句話。掏出小帳子，把你的名字、住處記下，走了。
你站在那裡，心裡想: 辯解，又能怎樣。
@set 反抗積累 += 2
:: else
你把揀好的幾朵花菜向前湊。
得參: 「大人，這些就送您，請笑收——」
巡查把菜攔住了。
巡查: 「汝這是做什麼？賄賂警察嗎？！」
那菜還捧在你手上，你一時不知道該怎麼辦。
巡查: 「拿過來！」
他接過秤仔，翻看了幾下，然後啪的一聲打斷，擲在地上，掏出小帳子——這次他把你的名字、住處、以及「試圖行賄」，通通記了下來，氣憤憤地走了。
得參(心聲): 「送菜……連送菜也是錯的。」
@set 反抗積累 -= 1
:: endif
@hide 巡查 fade=400
@wait 1000
你站在擔邊，失神地看著地上那兩截斷掉的秤仔。
那是鄰居借給你的。那是新新的一桿秤。
@wait 1500
幾個閒人湊過來，一個有年紀的搖著頭說。
旁人: 「該死的東西，到市上來，只這規矩也不懂？你說幾斤幾兩，難道他的錢你還敢拿嗎？」
得參: 「難道我們的東西，該白送給他的嗎？」
旁人: 「唉，你不曉得他的厲害，你還未嘗到他青草膏的滋味。」
得參: 「什麼？做官的就可任意凌辱人民嗎？」
旁人: 「硬漢！」
眾人議論了一回，也就散去了。
@set 秤仔被沒收 = true
