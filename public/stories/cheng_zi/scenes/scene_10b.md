你把手裡握著的東西，放了回去。
你坐到桌邊，天光一點一點地亮起來。爆竹聲遠遠的，斷了又響，響了又斷。
新年到了。
@wait 1500
@show 妻子 center normal
妻子: 「參仔，開正了，起來——」
她看見你坐在那裡，愣了一下。你抬起頭，用力地點了頭。
得參: 「我起來了。」
@hide 妻子 fade=400
@wait 1000
明天，你還是挑起擔子去。見到巡查，你還是陪笑，把最好的菜揀出來放在他面前，不敢提秤仔。
被刁難了，你咬牙。被罵了，你低頭。

:: if 面對罰金 == "硬吞"
得參(心聲): 「就這樣。想辦法。繼續。」
這三個字，你說了很多遍了。
:: elif 面對罰金 == "遺言"
得參(心聲): 「怨嘆無路用。阿母是對的。」
但這句話說出口的時候，是什麼滋味，你說不清楚。
:: endif
日子就這樣一天一天繼續。

:: if 金花未贖
有一天，那根金花終於贖回來了。

:: if 金花是阿秀的
阿秀把它拿在手裡，看了很久，沒有說話。然後放進抽屜裡，輕輕關上。
:: else
你讓阿秀送回去給她嫂嫂，說: 「謝謝她，讓我們借了這麼久。」
:: endif
:: endif
有時候夜深了，你看著睡著的孩子，心裡會有一種說不清楚的空洞。
得參(心聲): 「這樣還算是『活著』嗎？還是只是尚未死去？」
你沒有答案。你只是繼續。
@wait 2000
