@show 妻子 center worried
你在室內踱來踱去。阿秀幾次催你去睡，你都沒有聽見。
心裡只在想，總覺有一種不明瞭的悲哀，只不住漏出幾聲嘆息。
得參(心聲): 「人不像個人，畜生，誰願意做。這是什麼世間？活著倒不若死了快樂。」
你喃喃地獨語著，忽又回憶到母親死時，臉上快樂的容貌。
那是一種完成了的表情。
@hide 妻子 fade=400
@wait 1500
窗外，爆竹聲陸陸續續。鄰家在圍爐，笑聲隱隱傳來。
你想到了：
那桿斷掉的秤仔。

:: if 面對罰金 == "想搶"

:: endif

:: if 埋伏筆_每次硬吞

:: endif

:: if 金花未贖
那根還沒贖回來的金花——不知道阿秀心裡是怎麼想的。
:: endif
阿秀和孩子，在裡房裡。
@wait 2000

:: if 反抗積累 >= 6

:: if 童年印象 == "憤怒"

:: elif 埋伏筆_喪母憤怒

:: elif 埋伏筆_曾想反抗
也許是在市場上，你握緊了拳頭，又鬆開的那一次。
:: else
你不確定。但你知道它一直在那裡，從未真的熄滅過。
:: endif
這麼多年，你一直在壓著它。
今晚，你不想再壓了。
你懷抱著最後的覺悟。
@wait 2000
