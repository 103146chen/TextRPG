你決定做生菜的生意。
但問題是本錢。
你心地坦白，不敢向人開口借——但不借就沒有法子。
上哪裡去借這幾塊錢？
@set 上當鋪 = false

?? choice

:: label after_borrow

:: if 借錢對象 == "岳家"
@set 上當鋪 = true
阿秀回了娘家。
@show 妻子 center worried
一個小農民的妻子，哪有闊的外家——得不到多大幫助，本是情理中的事。但難得她嫂嫂待她還好，把自己唯一的裝飾品——一根金花髮簪——借給她，教她去當鋪裡押幾塊錢，暫作本錢。
妻子: 「嫂嫂說，等我們有錢了，要快快贖回來。那是她的心愛之物。」
得參: 「放心，我們一定快快贖回來。」
你握著那根金花，心裡很不是滋味。這錢，不是借來的——是嫂嫂的信任。
@hide 妻子 fade=400
@set 反抗積累 += 0
@set 金花未贖 = true
@set 借了外人心愛之物 = true
:: elif 借錢對象 == "廟口"
@show 老頭 center normal
你去廟口找了老頭家。
他在廟前擺了幾十年的香燭攤，是全村有名的厚道人。
頭家: 「你要借幾塊？生意本錢，可以的，可以的。」
他從抽屜裡掏出幾個銀元，按在你手心，只說了一句：
頭家: 「記得還就好，什麼時候都行。」
你站在那裡，說不出話，只低低地說了一聲謝。
得參(心聲): 「這個人情，沒有辦法用錢還的。」
@set 反抗積累 += 1
@set 欠廟口人情 = true
@hide 老頭
:: elif 借錢對象 == "典當"
@set 上當鋪 = true
家裡沒有什麼值錢的東西。
你翻找了一遍，只找到一根金花——那是阿秀出嫁時娘家給的，也是她唯一的首飾。
@show 妻子 center worried
妻子: 「……你要拿這個去當？」
得參: 「只是暫押，等賺到錢馬上贖回來。」
她沒有說什麼。你看見她把金花從頭上取下來，放進你手心的時候，那個動作很緩、很輕——像是在放走一件不想放走的東西。
@hide 妻子 fade=400
@set 反抗積累 -= 1
@set 金花未贖 = true
@set 金花是阿秀的 = true
:: else
@show 頭家 center normal
你去找曾經雇用過你的頭家，說了借支工錢的事。
頭家看了你半晌，叫你進去坐。
頭家: 「你以前做事勤快，這個忙幫得。」
他預支了你幾塊工錢，但說了一句話：
頭家: 「但這幾塊，以後要從工資裡一點一點扣回來。你要繼續幫我做工。」
你點了頭。只是——你心裡有一個角落，覺得有些不對勁。
得參(心聲): 「這樣，算是賣了自己的時間去換錢。」
@set 反抗積累 -= 1
@set 欠頭家工錢 = true
@hide 頭家
:: endif
有了本錢，你去買了第一擔生菜。
才要出門，阿秀發現少了一桿秤仔。
@show 妻子 center worried
妻子: 「沒有秤仔怎麼辦？」
得參: 「要買一桿，可是官廳的專利品，不是便宜的東西，哪兒來的錢？」
妻子: 「我去隔壁借一桿。」
@hide 妻子 fade=400
鄰居好意，借了一桿還新新的秤仔。你掂了掂，放進擔子裡。
得參(心聲): 「這一次，一定要好好賺，把本錢還清。」
你挑起擔子，走向鎮上。
@set 秤仔已借 = true
